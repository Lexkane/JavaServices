PROJECT DESCRIPTION  
Mortal Combat: Droid Edition  

Models (droids): Droid (abstract), SideSwipe, K2So, Healer, Ultron, Striker, Neo.  

Controllers:  
- MenuController – show droids info and send them to the arena.  
- ArenaController – logic of the fight (get winner, cooldowns, droids status, fights history).  

GUI: JavaFX.
